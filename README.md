Close CRM API Wrapper Service
=============================

A Close CRM API wrapper using Guzzle.

By [Simon Hampel](https://twitter.com/SimonHampel).

Installation
------------

The recommended way of installing the wrapper is through [Composer](http://getcomposer.org):

    :::json
    {
        "require": {
            "hampel/close": "~2.0"
        }
    }
