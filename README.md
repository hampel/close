Close.io Service
================

A Close.io API wrapper using Guzzle v6.

By [Simon Hampel](http://hampelgroup.com/).

Installation
------------

The recommended way of installing Close.io Service is through [Composer](http://getcomposer.org):

    :::json
    {
        "require": {
            "hampel/closeio": "~1.0"
        }
    }
